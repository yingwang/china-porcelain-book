# 第 6 章 · 质量控制

## 6.1 一千万片瓷片的考古证据

景德镇御窑厂遗址，从二十世纪七十年代起进行过数十次随工发掘和三次大规模主动发掘。出土的瓷片数量是惊人的——**数十吨、近一千万片** [^36]。

这一千万片不是完整器物，是被有意打碎的次品。它们大致按朝代分层堆积——明初洪武到永乐的瓷片在最下，宣德到正德的在中间，嘉靖、万历的在更上层，清代各朝的在最上。这是 542 年御窑厂运转留下的垂直时间剖面。

打碎的方式有规律可循。同一个器型，往往可见上百件碎片，但流传到宫廷或后世的只有一两件。剩下的几十、上百件都被故意敲碎、混杂掩埋。考古队员从堆积层里复原出整器需要把成百上千的碎片一一比对、拼合。一件被复原的明永乐青花大盘，有时是从几个不同探方、上千片碎片中拼出来的。

这种考古证据让"次品销毁制度"从一句话变成了一千万片瓷片的实物。

御窑厂的窑炉位置在 542 年间也有迁移。洪武到永乐时期的窑炉区在珠山北麓，使用葫芦形窑；宣德到万历时期窑炉区转移到珠山南麓西侧，主要使用馒头形窑 [^37]。这种迁移本身反映了工艺的更替——葫芦窑适合明初的小批量精烧，馒头窑适合宣德起更大规模的官窑生产。

一个具体的考古发现可以说明这种品控严苛的程度：1993 年春，景德镇市政府在中华路盖房，景德镇陶瓷考古研究所在明御器厂东门故址附近开探沟，在距地表约 1.5 米深的宣德窑渣层中发现一窝状堆积的青花残片。修复拼合后，这是一组**宣德蟋蟀罐**——圈足与盖内底都有"大明宣德年制"的单行青花楷书款；上层蟋蟀罐绘有双角五爪龙纹。按明代制度，只有皇帝可以使用五爪龙纹饰，所以这批罐确认为宣德皇帝御用之物 [^37]。

宣德皇帝以喜好斗蟋蟀著称（民间称"促织皇帝"），但奇怪的是清代宫廷收藏中几乎没有宣德蟋蟀罐传世。这批 1993 年出土的残片是迄今唯一可信的宣德官窑蟋蟀罐实物——它们都被有意打碎、掩埋在御窑厂的销毁层里。次品销毁制度在这件事上做到了极致：连皇帝爱好的器物，但凡不合格也一并销毁，不流入民间。

## 6.2 永乐朝的年款：一种品牌身份认证

景德镇瓷器在明代以前大多不带年款 [^38]。瓷器底部光素，或有简单的工匠名字，或有简单的吉祥图记，但没有"哪个朝代谁烧的"这种明确的身份信息。

明代永乐年间（1403–1424）开始，景德镇官窑首次系统性地在器物上写帝王年号款。永乐朝的年款只有四字——**"永乐年制"**——而且数量很少。永乐官窑器物中带年款的比例不高，多数仍是无款官窑。

到了宣德朝（1426–1435），年款制度才真正成为定制：

- **六字双行楷书年款**："大明宣德年制"
- **写在器物底部**或近底部
- **以青花书写**为主，兼有暗刻、凸印
- **多以双重圆圈或方框**围绕款字
- **有专门的写款工种**

宣德年款的密度大幅提升——传世宣德官窑器绝大多数都带款，这与永乐"少款"的风格形成鲜明对比。民间因此有一句总结："**永乐款少、宣德款多、成化款肥、弘治款秀、正德款恭、嘉靖款杂**" [^39]。每朝官窑款识的字体风格、位置、密度都各不相同，是后世断代的重要依据。

为什么宣德朝把年款做成定制？这有两个工程层面的解释：

一，**身份认证**。年款让每件器物都明确标注产地、时代、官窑身份。这是早期的品牌认证机制——一件带"大明宣德年制"款的瓷器，意味着它经过了御窑厂的工艺标准、监工核验、款识工序。这种认证让宫廷在分发、赏赐、礼制使用时能精确追溯。

二，**反向的责任绑定**。年款不只标"谁制造"，也标"谁负责"。一件带款官窑器如果质量有问题，可以追溯到具体朝代、具体督陶官、具体窑场。这种制度上的可追溯性，是任何成熟工业品控的基础。

到了清代，年款制度延续并细化。康熙、雍正、乾隆三朝官窑款识各有规范，写款工艺极为讲究，部分御窑器的款字本身就是一件书法作品。

## 6.3 多层级监督

御窑厂的品控不是一道工序，是多层级的连续把关：

**第一层：窑工的自检**

每件器物在拉坯、上釉、画款、装匣阶段都由对应工种的师傅做自检。明显的开裂、变形、色差在进入窑炉前就被剔除。这一层剔除率不公开，但根据后世类似手工业的经验，约 5%–15% 的坯体在装窑前就被弃。

**第二层：开窑后的初检**

烧成开窑后，景德镇御窑厂有专门的"选拣房"。每件出窑器物由经验丰富的师傅初步检视，明显的烧成缺陷（流釉、裂痕、色不正、变形、款识写错）被剔除。这一层剔除率根据出土堆积层估算约 50%–80%，因器型和时代而异。

**第三层：监工核验**

通过初检的器物再交由督陶官或他指定的监工核验。这一层核验更严苛——不只看物理缺陷，还看是否符合朝廷下达的图样规格、款识是否标准、釉色是否符合宫廷审美。这一层剔除率约 30%–50%。

**第四层：京师终验**

通过景德镇监工核验的器物装船运到京师，进入紫禁城后由内务府或工部再做一次终验。这一层主要检查运输损伤和宫廷端的审美判断，剔除率较低（约 5%–10%）。

把这四层乘起来，从入窑坯体到最终入宫的转化率大约是 5%–20% 之间——也就是说，烧 100 件入宫的，可能要烧 500 到 2000 件入窑坯体。良品率在工艺最严苛的品种（如祭红、汝窑式青瓷）上更低，可低至 1%–5%。

这套多层级品控体系的物质遗存，就是御窑厂遗址那一千万片碎瓷。

## 6.4 销毁制度的工程意义

为什么次品要销毁，不能流入民间？这一点对现代读者有时显得浪费——为什么一件烧得不够好、但仍然能用的瓷碗，要被打碎掩埋？

理由有三个层面。

**层面一：品牌保护**

如果次品流入民间，民间会出现"略低品质的官窑器"。这会模糊官窑与民窑的边界，让"大明宣德年制"这个标签失去含金量。销毁次品是保护这个标签的稀缺性。

**层面二：礼制等级**

中国前现代社会有严格的礼制等级。某些器型、某些纹饰（比如五爪龙、特定颜色的黄釉）只有皇帝或特定品级官员才能使用。如果带这些规格的次品流入民间，等于打破了礼制——一个百姓家中出现五爪龙纹瓷碗，是逾越。销毁是维护这一等级。

**层面三：工艺保密**

次品上仍然带有官窑的全套工艺特征——配方、釉色、画法、款识。如果次品被民窑拿去研究、拆解、仿制，官窑的工艺就泄露了。销毁是工艺保密的一种粗暴但有效的方式。

这三层理由今天看仍能找到对应——奢侈品行业的次品销毁、医药行业的过期药销毁、半导体行业的有缺陷晶圆销毁，逻辑链条几乎一样。

## 6.5 一千年的"金字塔形产能"

把品控制度和良品率合起来看，景德镇官窑的整体产能可以画成一个金字塔：

```
       ▲ 入宫成品
      / \  (~5-20%)
     /   \
    / 监工 \
   /  核验  \  (~30-50%)
  /          \
 / 选拣房初检 \  (~50-80%)
/              \
================
窑工自检 + 入窑（100%）
```

每一层的损耗都对应一座销毁堆。每一千件入窑坯体，最终只有 50–200 件入宫；其余八九百件都化作御窑厂遗址的瓷片堆积层。

这种金字塔产能在现代制造业的视角看是低效的——半导体行业即使最严苛的高端芯片良品率也通常在 50% 以上，普通日用品 95%+。但景德镇官窑的目标不是效率，是**质量上限**。它代表的是工业系统在不计成本前提下能达到的工艺极限。

有意思的是，这种"为了上限不计成本"的逻辑在现代仍然存在——奢侈品手表的钟表师手工组装、F1 赛车的引擎装配、太空仪器的航天级元件——都是同样的逻辑。明清御窑厂是这种逻辑的早期版本，规模化运作了五百多年。

## 6.6 一千万片瓷片今天还在做什么

御窑厂遗址近年继续在做考古工作。2021 年发布了《景德镇御窑厂西围墙遗址考古发掘报告》（一），整理宋元窑业与明清官窑遗存 [^40]。出土的瓷片不仅是历史证据，也是工艺研究的活样本——研究者可以从碎片的胎、釉、画工细节中推算明清各朝的工艺参数变化。

2023 年起，景德镇启动"千万陶片身份证"项目——用数字化手段对每一片有信息的瓷片建档、扫描、分类。目标是把这一千万片建立可检索的数据库，让全世界研究者能远程访问 [^41]。这是中国陶瓷考古从纸本档案走向数字资产的一步。

这一千万片瓷片的意义至此完成了一次反转：它们最初是工业系统**销毁次品**的产物——是要被掩埋、被隔离、被忘记的；六百多年后，它们成为**研究这个工业系统**最重要的物质证据。

下一章我们看一个看似简单但关键的问题：这套高品控、严销毁的工业系统，如何在几十万件甚至几百万件级别的产量上保持标准化？这是现代工业的核心问题，景德镇用什么方式回答它？
