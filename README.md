# 瓷：中国第一个工业

> 把中国瓷器从"工艺美术"重新放回"工业史"。

**Languages**: [中文](README.md) | [English](en/README.md)

景德镇从唐末到清末连续千年的生产，本质是人类历史上最早的、最完整的、规模最大的工业系统。它解决的所有问题——原料、能源、标准化、质量控制、分工、物流、海外销售、知识传承、被反向工程——和今天硅谷或台积电解决的问题是同一类问题。

这本书把瓷器放回它真正的位置：**工业**。

## 引用与考据

这本书的每一句涉及具体年代、数量、人物、事件的陈述，都附有可追溯的公开来源——考古发掘报告、博物馆官方资料、学术期刊论文、权威媒体记录、地方志。每章末尾设独立的"参考文献"小节，章内引用编号连续延伸（第 1 章 [^1]–[^15]，第 2 章 [^20]–[^24]，第 3 章 [^16]–[^19] 等）。

凡未在手边有可靠出处的细节，宁可不写，也不靠想象补全。模糊带过的地方明确标"下一稿需查"作为研究 todo。这本书的最低底线是：**每一句已经写出来的话都对，每一处对的事都能查证。**
